# ⚽ Voltixai Infosport - Publication Automatique

Robot **gratuit et automatique** qui publie en direct les matchs, les buts et les résultats sur la page Facebook **Voltixai Infosport**.

---

## 📋 Ce que fait le robot
- 📅 **Annonce des matchs à venir** : équipes, heure et compétition
- ⚡ **Mises à jour en direct** : évolution du score, minute du match
- 🥳 **Alerte buts** : message spécial quand un but est marqué
- ✅ **Résultats finaux** : score final à la fin de la rencontre
- 🕒 **Heure locale** : affichage en heure de Brazzaville (GMT+1)

---

## 🛠️ Outils et sources
- **Données des matchs** : API AnySport
- **Publication Facebook** : Graph API v25.0
- **Exécution automatique** : GitHub Actions
- **Fréquence** : Vérification toutes les 3 minutes

---

## 🚀 Lancement
Le robot fonctionne **tout seul** sans intervention.
Pour un test manuel :
1. Aller dans l'onglet **Actions**
2. Choisir `Publication Voltixai Live Foot`
3. Cliquer sur **Run workflow**
